# Projeto PAP - Site de Aprendizado de Música

Este projeto é um site desenvolvido para a Prova de Aptidão Profissional (PAP), com o objetivo de ajudar pessoas a aprender música.

## Configurações do Ambiente

* **Ambiente Virtual:** Foi criado um ambiente virtual (`.venv`) para isolar as bibliotecas do projeto.
* **Como ativar:** No terminal do Windows, utilize o comando:
  ```bash
  .\.venv\Scripts\Activate.ps1

Bibliotecas e Extensões
SQLite Viewer: 
Extensão instalada no VS Code para visualizar os dados do arquivo usuarios.db diretamente no editor, sem a necessidade de programas externos.

Bibliotecas Python: 
Todas as dependências necessárias estão listadas no arquivo requirements.txt.

Banco de Dados
O banco de dados utilizado no projeto é o SQLite.

Arquivo: 
usuarios.db

Como visualizar: Basta clicar no arquivo dentro do VS Code (após instalar a extensão citada acima) para ver as tabelas.

O Quiz
O quiz foi desenvolvido com base em pesquisas de outros quizzes de música e após entrevistas com amigos que são professores da área, garantindo perguntas didáticas.

Instrumentos
Os instrumentos foram escolhidos com base no meu conhecimento pessoal, sendo estes os que eu já sei tocar. No futuro, o projeto poderá ser expandido com a adição de novos instrumentos.
