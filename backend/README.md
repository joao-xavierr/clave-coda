# Projeto PAP - Site de Usuários
# MARKDOWN

Este projeto é um site que ajuda as pessoas a aorender música desenvolvido para a PAP.

## Configurações do Ambiente
- **Ambiente Virtual:** Foi criado um `.venv` para isolar as bibliotecas.
- **Como ativar:** No terminal, use `.\.venv\Scripts\Activate.ps1`.

## Bibliotecas e Extensões
- **SQLite Viewer:** Instalei esta extensão no VS Code para visualizar os dados do arquivo `usuarios.db` sem precisar de programas externos.
- **Bibliotecas Python:** Estão listadas no arquivo `requirements.txt`.

## Banco de Dados
O banco de dados utilizado é o **SQLite**.
- Arquivo: `usuarios.db`
- Para ver as tabelas, basta clicar no arquivo no VS Code após instalar a extensão citada acima.

## O QUIZ
O quiz foi feito pensado nas perguntas de outras quizzes e após perguntas a amigos que são professores de música.
